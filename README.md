### Requirements
Node v20.
### Installation
##### 1) Clone the project.
```
Git clone https://github.com/CarSeP/Url-Shortener.git
```
##### 2) Install dependencies.
```
npm i
```
##### 3) Run the project.
```
npm run dev
```
